Listings Demo

Nikola intends to let you show code easily via listings:

hello.py

#!/usr/bin/python

import sys


def hello(name='world'):
    greeting = "hello " + name
    print(greeting)

if __name__ == "__main__":
    hello(*sys.argv[1:])
